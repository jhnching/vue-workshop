What is Vue.js

  • View layer only
  • Uses a virtual DOM similar to React
  • Uses watchers to determine what content to re-render
  • Templating is similar to Angular

Vue.js Component Architecture

Data

Contains all the data scoped within the component for use


          export default {
            name: 'App',
            data: function() {
              return {
                classId: 'class1',
                students: []
              }
            }
          };

Computed

An object containing functions that perform calculations on data and will auto update whenever component properties change. Relies on caching. Computed method accessed similar to a data prop. 


          export default {
            name: 'App',
            data: function() {
              return {
                message: 'abc'
              }
            },
            computed: {
              upperCasedMessage: function() {
                return this.message.toUpperCase();
              }
            }
          };

Methods

An object containing functions that provide functionality to the app. Unlike computed properties, methods will always re-evaluate. 


          export default {
            name: 'App',
            methods: {
              toUpperCase: function(value) {
                return value.toUpperCase();
              }
            }
          };
